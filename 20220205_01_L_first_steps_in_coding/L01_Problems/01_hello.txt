Напишете функция, която отпечатва текста "Hello SoftUni".

Насоки:
------
    *) Създайте нов JavaScript файл в съществуващата папка и го именувайте подходящо.
       Препоръчително е всеки скриптов файл да се казва както името на задачата, чието решение съдържа.


    *) Съдържанието на новият файл ще се отвори в прозореца в дясно.


    *) Отидете във файла hello.js и създайте функцията hello(). Можете да си помогнете с картинката по-долу:

function hello() {
  console.log("Hello SoftUni");
}


    *) Стартирайте програмата с: Ctrl + F5. За да получим резултат трябва да "извикаме" функцията:

function hello() {
  console.log("Hello SoftUni");
}

hello();
